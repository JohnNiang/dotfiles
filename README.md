# Dotfiles

## Copy commands

### i3 

```shell
cp ~/.config/i3/config .
```

### i3status

```shell
cp ~/.config/i3status/config . // copy i3status config
cp ~/.config/i3status/net-speed.sh . // copy net speed shell script, you have to execute <chmod +x net-speed.sh> before
```

### zsh

```shell
cp ~/.zshrc .
```

### redshift

```shell
cp ~/.config/redshift.conf .
```
